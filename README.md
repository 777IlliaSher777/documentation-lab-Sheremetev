# HelpDesk Pro

**Коротко про систему**  
HelpDeskPro — це проста веб‑система підтримки користувачів для малих команд і стартапів, яка дозволяє приймати, маршрутизувати та відстежувати звернення (тикети) від клієнтів або внутрішніх користувачів. Система централізує запити, уніфікує процеси обробки, дозволяє відстежувати статуси, призначати виконавців, додавати коментарі та вкладення, а також зберігати повну історію взаємодій для прозорого та ефективного вирішення проблем.

**Проблема, яку вирішує**  
Система зменшує час реакції на звернення, уніфікує процеси підтримки та зберігає історію взаємодій, що допомагає уникати дублювання роботи та підвищує якість обслуговування. Замість хаотичних листів і повідомлень у месенджерах HelpDeskPro збирає всі звернення в одному місці, допомагає пріоритезувати задачі, автоматизовано призначати виконавців і пришвидшує їхнє вирішення.

**Основні функції**
- Прийом звернень через веб‑форму або email; централізація всіх запитів у єдиному інтерфейсі.
- Створення та управління тикетами: додавання, редагування, присвоєння статусів (Open, In Progress, Resolved, Closed) і пріоритетів (low, medium, high).
- Маршрутизація та призначення: автоматичне або ручне призначення тикетів на команду або конкретного агента.
- Комунікація в тикетах: коментарі від підтримки та клієнта, вкладення файлів, історія переписки.
- Пошук, фільтрація та нотатки: швидкий пошук за тегами, фільтри за статусом/пріоритетом/категорією; нотатки з тегами для внутрішніх коментарів.
- Нагадування та дедлайни: сповіщення про терміни, SLA‑тригери та нагадування агентам і запитувачам.
- Профіль користувача: налаштування, персоналізація, керування сповіщеннями.
- Аналітика та звіти: панель показників (SLA, середній час вирішення, навантаження агентів).
- Простий REST API для інтеграцій: ендпоінти для створення/отримання/оновлення тикетів і користувачів.

**Для кого цей продукт**
- Служби підтримки малих і середніх компаній.
- Внутрішні IT‑команди для обробки інцидентів.
- Проєктні менеджери для відстеження запитів клієнтів.
- Фриланс‑команди для централізації звернень.
