# HelpDesk Lite

HelpDesk Lite — проста система підтримки користувачів для малих команд і стартапів. Централізує звернення, дозволяє відстежувати статуси, призначати виконавців і зберігати історію взаємодій.

**Проблема, яку вирішує**  
Замість хаотичних листів і повідомлень у месенджерах HelpDesk Lite збирає всі звернення в одному місці, допомагає пріоритезувати задачі та пришвидшує їхнє вирішення.

**Основні функції**
- Прийом звернень через форму або email  
- Присвоєння статусів і пріоритетів  
- Коментарі від підтримки та клієнта  
- Пошук і фільтрація звернень  
- Простий REST API для інтеграцій

**Цільова аудиторія**
- Малий бізнес і стартапи  
- Команди техпідтримки без виділеного helpdesk-інструменту  
- Навчальні проєкти та демонстрації

**Структура репозиторію**
