# HelpDesk Pro

**Коротко про систему**  
HelpDeskPro — це проста веб‑система підтримки користувачів для малих команд і стартапів, яка дозволяє приймати, маршрутизувати та відстежувати звернення (тикети) від клієнтів або внутрішніх користувачів. Система централізує запити, уніфікує процеси обробки, дозволяє відстежувати статуси, призначати виконавців, додавати коментарі та вкладення, а також зберігати повну історію взаємодій для прозорого та ефективного вирішення проблем.

**Проблема, яку вирішує**  
Замість хаотичних листів і повідомлень у месенджерах HelpDesk Lite збирає всі звернення в одному місці, допомагає пріоритезувати задачі та пришвидшує їхнє вирішення.

**Основні функції**
- Прийом звернень через форму або email  
- Присвоєння статусів і пріоритетів  
- Коментарі від підтримки та клієнта  
- Пошук і фільтрація звернень  
- Простий REST API для інтеграцій

**Цільова аудиторія**
- Малий бізнес і стартапи  
- Команди техпідтримки без виділеного helpdesk-інструменту  
- Навчальні проєкти та демонстрації
